# HyperFrames Workspace

This folder is an isolated HyperFrames workspace for previewing and rendering HTML compositions without touching the main Lifemode app.

Current status:
- HyperFrames CLI setup only
- No composition content added yet
- Local render uses static FFmpeg/FFprobe shims from this workspace

Use:
- `npm run hf:preview`
- `npm run hf:render`
